# partitura

### Create a venv

```sh
python3 -m venv venv
```

### Activate the venv
```sh
. venv/bin/activate
```

### Install requirements.txt
```sh
pip install -r requirements.txt
```


### Set FLASK variables:
Development mode
```sh
export FLASK_APP=app
export FLASK_ENV=Development
export FLASK_DEBUG=True
```

### Run app:
Development mode

```sh
python3 run.py
```
